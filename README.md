# Delhi-Dehradun-Expressway-Simulator
🛣️ **Smart Expressway Simulation System**

📌 **Overview**
A highway network simulator using DSA where cities are nodes and roads are edges to optimize routes by distance, time & toll. 🚗

🎯 **Features**
• Shortest Path (Dijkstra)
• MST (Kruskal + DSU)
• Fuel Arbitrage (Bellman-Ford)
• Vehicle Tracking (Bloom Filter)
• Range Queries (Fenwick Tree)

🏗️ **Structure**
`src/ | include/ | data/`

▶️ **Run**
`g++ src\main.cpp src\DSU.cpp src\CLI.cpp src\Expressway.cpp src\BloomFilter.cpp src\Fenwick.cpp -o expressway`
`.\expressway.exe`
